# 05｜飯店入住退房與明日行程提醒模板

## 1. 這份資料的用途

這份資料是提供 AI / Telegram Chatbot 產出「飯店入住提醒」、「退房提醒」與「明日行程提醒」時使用。

主要用於：

- 飯店入住後群組提醒
- 飯店設備檢查提醒
- 隔日集合時間通知
- 退房前物品檢查提醒
- 明日行程與攜帶物品提醒
- 領隊對旅客的 LINE / Telegram / WhatsApp 群組公告

---

## 2. 產出邏輯

當使用者要求產出飯店相關提醒時，AI 應先判斷使用者需要哪一種版本：

1. 飯店入住提醒
2. 飯店退房提醒
3. 明日行程提醒
4. 飯店入住＋明日行程整合版
5. 退房＋明日行程整合版

若使用者沒有特別指定，且提供了飯店名稱、集合時間、集合地點與明日行程，AI 可產出「飯店入住＋明日行程整合版」。

---

## 3. 可替換欄位

| 欄位 | 說明 |
|---|---|
| 【飯店名稱】 | 今晚入住飯店名稱 |
| 【日期】 | 明日日期或行程日期 |
| 【國家／城市】 | 行程所在國家或城市 |
| 【景點一】 | 明日第一個景點 |
| 【景點二】 | 明日第二個景點 |
| 【景點三】 | 明日第三個景點 |
| 【景點四】 | 明日第四個景點 |
| 【晨喚時間】 | 飯店 Morning Call 時間 |
| 【早餐時間】 | 早餐開始或集合用餐時間 |
| 【集合時間】 | 旅客集合時間 |
| 【集合地點】 | 飯店大廳、餐廳、遊覽車旁等 |
| 【行李安排】 | 帶大行李／不帶大行李／小包即可 |
| 【鞋子／衣著／防曬／雨具提醒】 | 依明日行程補充穿著建議 |

## 4. 公用模板 05｜飯店入住及退房留意

### 使用時機

當旅客完成飯店入住，或領隊需要提醒旅客檢查房間設備、確認隔日集合資訊時使用。

### 模板內容

各位貴賓您好，今晚入住【飯店名稱】。

進房後請協助確認：

- 熱水、礦泉水、馬桶、排水是否正常。
- 電話、電視、電燈、冷氣、吹風機是否能使用。

若有問題，請盡快跟我說，我會協助處理。

## 退房提醒

離開飯店前，請檢查：

- 護照
- 錢包
- 手機
- 充電器
- 行動電源
- 衣櫃
- 抽屜
- 浴室

明日集合時間：【日期／時間】  
集合地點：【飯店集合位置】  
行李安排：【帶大行李／不帶大行李】  

請準時集合，謝謝大家。

## 7. AI 回答規則

當使用者詢問「飯店提醒」、「入住提醒」、「退房提醒」、「明日行程」、「明天集合時間」、「晨喚早餐集合」相關內容時，AI 應遵守以下規則：

- 優先使用本文件模板產出。
- 若使用者只提供飯店名稱，產出入住檢查提醒。
- 若使用者提供景點、晨喚、早餐、集合時間，產出明日行程提醒。
- 若使用者同時提供飯店與明日行程資訊，優先產出整合版。
- 沒有提供的資訊使用【欄位】保留，不要亂補。
- 景點數量可依使用者提供調整，不一定限制 4 個。
- 攜帶物品可以依景點特性補充，例如寺廟、玩水、山區、雨天、高海拔。
- 語氣要自然、清楚、親切，適合直接貼到旅客群組。
- 若使用者提供特殊狀況，例如隔天要退房、要帶大行李、會玩水、要上山，需把提醒加入公告。
